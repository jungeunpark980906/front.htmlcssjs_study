배열관련 메서드 : 유사(연관)배열 에서는 사용제한이 있음

1. join() : 배열 원소 사이에 문자를 삽입하여 새로운 문자열을 생성 2. push(value) : 배열의 마지막에 value를 추가 => push 후 배열의 길이를 리턴 (array[array.length]=value;와 동일 효과) 3. unshift(value) : 배열의 가장 앞에 원소를 추가 => 추가 후 배열 길이 리턴 4. pop() : 배열의 마지막 원소 제거 => 제거한 원소의 값 리턴 5. shift() : 배열의 첫 원소를 제거 => 제거한 원소의 리턴
